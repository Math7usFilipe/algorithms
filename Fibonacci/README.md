### Fibonacci Sequence
**Problema:**
Dado um número 'n', o algoritmo deve encontrar os 'n' elementos da sequência de Fibonacci.
<br>
**Fibonacci Sequence:***
A sequência de Fibonacci é uma série numérica em que cada número é a soma dos dois anteriores, começando com 0 e 1.
<br>
**Exemplo:**
<br>
Fibonacci(2) = [0, 1]
<br>
Fibonacci(3) = [0, 1, 1]
<br>
Fibonacci(4) = [0, 1, 1, 2]
